Key Observations

  • MediaStream stays open continuously ‚Üí causes persistent recording indicator
  • Old ImageBitmap not closed when React state is updated ‚Üí potential memory leak
  • Full tab captured (~8MB) but only small portion displayed in 400√ó400 lens
  • Original bitmap is properly closed after cropping
  • Fisheye path adds 2 extra canvas copies before final display
  • Each recapture (scroll/resize/DOM change) repeats the full capture cycle
