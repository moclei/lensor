# Lensor

A Chrome extension that overlays a draggable, zoomable magnifying lens on any webpage for pixel-perfect color detection and palette generation.

## About

Lensor lets you inspect any webpage at the pixel level. Drag the lens anywhere to magnify content, detect exact colors, and generate beautiful color palettes automatically.
